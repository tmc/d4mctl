# d4mctl

Command d4mctl controls a docker-for-mac instance.

```sh
$ d4mctl -h
Usage:
  d4mctl [command]

Available Commands:
  cpus        get or set number of cpus enabled in the local docker-for-mac installation
  dump        dump current configuration
  help        Help about any command
  k8s         get or set state of k8s-enabled in the local docker-for-mac installation
  mem         get or set amount of memory alllocated for the local docker-for-mac installation (in megabytes)
  restart     restart docker-for-mac

Flags:
  -h, --help   help for d4mctl

Use "d4mctl [command] --help" for more information about a command.
```
